Each role maps to a different LLM by design — exploiting the empirical strengths each model showed in Alpha Arena Season 1 (Oct–Nov 2025). The default path runs through OpenRouter on a single key; an Anthropic-only fallback substitutes Claude for every role.

What this demo does

  • Mock mode — deterministic stub responses, no API calls, instant. Free.
  • Real mode — runs the 5-stage pipeline through OpenRouter against the architecture-spec asymmetric LLM map. ~60s per cycle, ~$0.05. Hard cap of 10 real runs / UTC day.
  • Sample runs — 11 pre-saved real runs + a 10-run aggregate summary.

Pipeline (per cycle)

  1. 4 analysts in parallel — Fundamentals, Technical, News, Sentiment.
  2. Bull / Bear sequential debate over the analyst reports.
  3. Trader proposes a structured JSON trade.
  4. Risk Manager approves / modifies / vetoes against vault constraints.
  5. Fund Manager makes the final call → mock Tilt execution on chain 46630.

Why asymmetric LLMs?

The lazy submission is "Claude vs GPT vs Gemini head-to-head" — that's Alpha Arena, already done. Our move: use AA Season 1 results as labelled empirical data on each model's behavioural personality, then assign each LLM to the role where it demonstrated strength. Qwen3 Max (S1 winner, +22.3%, disciplined low-frequency execution) becomes the Trader. Claude Sonnet 4.6 (most cautious in S1 — wrong personality for a trader, exactly right for a risk officer) becomes the Risk Manager and the bull/bear/fund-manager stack.
